# v2ray

```sh
wget https://direct.io/go.sh -O /tmp/go.sh

# 安装 v2ray
sudo bash /tmp/go.sh
sudo bash /tmp/go.sh -c # check update

# 安装 v2sub
# pip3 install --user v2sub
sudo pip3 install v2sub
v2sub add http://xxxx.xx
v2sub update
v2sub list
cat ~/.v2sub/subscribes.json
cat ~/.v2sub/servers.json
```
